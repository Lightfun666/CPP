# scope

## variable in block
```cpp
#include <iostream>
using std::cout;
using std::endl;
int y = 1;
int main() {

    // x can be used only in while
    while(int x = ::y){
        if(::y > 3) ::y = -1;
        
        cout << "x=" << x << ", y="<< ::y++ <<endl;
    }
    
    // error: can to use x
    // cout << x;
    cout << "result y = " << y << endl;
    return 0;
}
```
output
```
x=1, y=1
x=2, y=2
x=3, y=3
x=4, y=-1
result y = 0
```
